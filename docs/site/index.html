How it works

1️⃣

Setup in 30 seconds

Run npx @aman_asmuei/aman to create your AI identity across 25 personality archetypes.

2️⃣

Pick a showcase

Run aman showcase rutin to install any of 13 real-world AI companions — medication reminders, business trackers, and more.

3️⃣

Deploy anywhere

Run on Telegram, Discord, or the web. One-click deploy to Railway or Render. Or run fully local with Ollama — no API key needed.

7-Layer Architecture

acore

Identity — who your AI is. 25 personality archetypes, git-versioned.

amem

Memory — semantic recall, knowledge graph, conflict detection.

aflow

Workflows — reusable multi-step processes your AI follows.

arules

Guardrails — what your AI will and won't do.

askill

Skills — domain expertise, levels up with use.

akit

Tools — 25 MCP-ready integrations (GitHub, Slack, databases…).

aeval
