# Getting Started with Aman Rutin

## For Yourself

### Step 1: Tell the bot your medications
```
You: I take ubat darah tinggi every morning after breakfast

Aman Rutin: Added! 💊 Ubat darah tinggi
            Time: Every morning, after breakfast
            Dosage: (What dosage? e.g., 5mg, 1 tablet)

You: 1 tablet 5mg

Aman Rutin: ✅ Updated: Ubat darah tinggi — 1 tablet 5mg, every morning after breakfast
            I'll remind you daily!
```

### Step 2: Add more items
```
You: I also take vitamin D at lunch and ubat kolesterol at night

Aman Rutin: Added:
            💊 Vitamin D — lunch time
            💊 Ubat kolesterol — night time

            Dosage for each?
```

### Step 3: Confirm when done
```
Aman Rutin: ⏰ Pagi! Dah ambil ubat darah tinggi?

You: Dah

Aman Rutin: ✅ Ubat darah tinggi — done!
            🔥 4 hari berturut-turut!
            Next: Vitamin D at lunch.
```

## For Your Parents (Setup by Anak)

### Step 1: Set up for them
```
You: Setup ubat for my mak. She takes:
     - Amlodipine 5mg morning
     - Metformin 500mg with lunch and dinner
     - Simvastatin 20mg at night

Aman Rutin: Setting up for Mak:
            💊 Amlodipine 5mg — every morning
            💊 Metformin 500mg — lunch & dinner
            💊 Simvastatin 20mg — every night

            Language for reminders: Malay or English?
```

### Step 2: Your parent just confirms
```
Aman Rutin: ⏰ Mak, dah ambil ubat pagi? (Amlodipine)

Mak: Dah

Aman Rutin: ✅ Bagus mak! Ubat pagi dah ambil.
            Nanti tengah hari ada ubat lagi ya.
```

## Daily Routine (Beyond Medication)
You can track anything daily:
- "Add: jalan pagi 30 minit setiap hari"
- "Add: check blood pressure every evening"
- "Add: minum air 8 gelas sehari"
- "Add: baca Quran selepas Subuh"

## Commands
| Say this | What happens |
|----------|-------------|
| "Add ubat [name]" | Add new medication |
| "Dah" / "Done" / "✅" | Confirm task completed |
| "Status" / "Hari ni" | See today's checklist |
| "Minggu ni" / "History" | See weekly adherence |
| "Setup for mak/ayah" | Set up for parent |
| "Tambah rutin [task]" | Add daily routine task |
