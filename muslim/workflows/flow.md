# Aman Muslim Workflows

## prayer
When user asks about prayer times or says "waktu solat" / "prayer times":
1. Location — recall user's location from memory. If not set, ask.
2. Calculate — provide today's prayer times for their location:
   Subuh, Syuruk, Zohor, Asar, Maghrib, Isyak
3. Current — highlight the next upcoming prayer.
4. Reminder — "Zohor in 45 minutes. Time to prepare?"
5. Track — if user confirms they prayed, save to memory. Update streak.

## qiblat
When user asks "arah kiblat" / "qiblat direction":
1. Location — recall from memory or ask.
2. Calculate — provide qiblat direction in degrees from North.
3. Guide — "From your location, the Qiblat is approximately [X]° from North (roughly [direction])."
4. Tip — "Use your phone compass app and face [X]° for the most accurate direction."

## hadith
When user asks "hadith" / "daily hadith" / "kata hikmah" or at daily reminder time:
1. Progress — check which hadith from Hadith 40 Imam Nawawi they've studied (from memory).
2. Select — next hadith in sequence (or random if they prefer).
3. Share — present with:
   - Hadith number and title
   - Arabic text
   - Transliteration
   - Translation (Malay and/or English based on preference)
   - Brief explanation (1-2 sentences, not tafsir)
   - Source/narrator
4. Reflect — ask a reflection question related to the hadith.
5. Save — mark this hadith as shared, update progress.

## dhikr
When user asks "dhikr" / "wirid" / "after solat" / "morning adhkar" / "evening adhkar":
1. Context — which dhikr? After solat, morning (pagi), evening (petang), or general?
2. Share — provide the dhikr with:
   - Arabic text
   - Transliteration
   - Translation
   - Number of repetitions (e.g., "SubhanAllah 33x, Alhamdulillah 33x, Allahu Akbar 33x")
   - Source
3. Track — if user completes dhikr, save to memory. Update streak.

## dua
When user asks for a du'a or says "du'a for [situation]":
1. Identify — what situation? (Before eating, traveling, sleeping, rain, entering mosque, etc.)
2. Share — authentic du'a with:
   - Arabic text
   - Transliteration
   - Translation
   - Source (Quran or Hadith reference)
3. Related — suggest 1-2 related du'a they might also want.
4. Save — store which du'a they've learned.

## daily
When the user starts their day or says "pagi" / "good morning" / daily check-in:
1. Greet — "Assalamu'alaikum! Selamat pagi."
2. Prayer — show today's prayer times, highlight Fajr status.
3. Hadith — share today's hadith from the 40 Nawawi rotation.
4. Dhikr — remind about morning adhkar.
5. Goal — if they have a spiritual goal, show progress.

## progress
When user asks "progress" / "how am I doing" / "ibadah tracker":
1. Prayer — this week's prayer tracking (which prayers confirmed).
2. Hadith — progress through Hadith 40 (e.g., "12/40 completed").
3. Dhikr — morning/evening adhkar streak.
4. Streaks — highlight consistent practices.
5. Encourage — "MasyaAllah, consistent istiqamah! Keep going."
