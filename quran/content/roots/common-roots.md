# 20 Most Common Quranic Roots

## How Roots Work
Arabic words derive from 3-letter roots. Each root carries a core meaning. By adding vowels, prefixes, and suffixes, one root produces many related words. Learning roots is the fastest way to build Quranic vocabulary.

---

## Root 1: ع-ل-م ('a-l-m) — Knowledge
- عَلِمَ ('alima) → he knew
- عِلْم ('ilm) → knowledge
- عَالِم ('aalim) → knowledgeable one/scholar
- عَلِيم ('Aleem) → All-Knowing (divine name)
- عَالَم ('aalam) → world
- تَعَلَّمَ (ta'allama) → he learned

---

## Root 2: ر-ح-م (r-h-m) — Mercy
- رَحِمَ (rahima) → he had mercy
- رَحْمَة (rahma) → mercy
- رَحِيم (Raheem) → Most Merciful (divine name)
- رَحْمَان (Rahman) → Entirely Merciful (divine name)
- أَرْحَام (arham) → wombs; pl. of rahim
- رَحَّامَة (rahhaama) → greatly merciful

---

## Root 3: ك-ت-ب (k-t-b) — Writing
- كَتَبَ (kataba) → he wrote
- كِتَاب (kitaab) → book/scripture
- كَاتِب (kaatib) → writer/scribe
- كِتَابَة (kitaaba) → writing (act of)
- مَكْتُوب (maktoob) → something written/destined
- كُتُب (kutub) → books (plural)

---

## Root 4: أ-م-ن (a-m-n) — Faith / Security
- آمَنَ (aamana) → he believed

- أَمَان (amaan) → safety/security

- اطمأَنَّ (itma'anna) → he found tranquility

## Root 5: ع-ب-د ('a-b-d) — Worship

- عَبِيد ('abeed) → servants (plural)
- عَابِد ('aabid) → worshipper
- مَعْبُود (ma'bood) → that which is worshipped

## Root 6: ق-و-ل (q-w-l) — Saying

## Root 20: ش-ك-ر (sh-k-r) — Gratitude
- شَكَرَ (shakara) → he was grateful
- شُكْر (shukr) → gratitude/thanks

- شَكُور (Shakoor) → Most Appreciative (divine name)
- كَفُور (kafoor) → ungrateful (antonym by contrast)
